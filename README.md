## Technologies utilisées:

-   Node.js
-   Express.js
-   EJS
-   Tailwind CSS
-   SQLite

## Installation:
Se placer dans le dossier du projet et exécuter la commande suivante :

```bash
npm install
```

## Lancer le projet:
Se placer dans le dossier du projet et exécuter la commande suivante :

```bash
npm run dev
```
